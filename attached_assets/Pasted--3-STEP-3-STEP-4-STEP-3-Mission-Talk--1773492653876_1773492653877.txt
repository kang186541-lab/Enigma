데일리 코스 3번째 메시지야. STEP 3(실전 말하기)과 STEP 4(복습 챌린지), 미션 완료 화면을 구현해줘.

## STEP 3: 실전 말하기 (Mission Talk)

이 STEP이 가장 중요해. GPT-4o와 상황별 대화를 하면서 최소 8~10문장을 말하는 단계야.

### 화면 구성

```
┌─────────────────────────────┐
│  Day 1 · 만남 인사            │
│  STEP 3/4 실전 말하기         │
│  ██████░░                    │
├─────────────────────────────┤
│                              │
│  🎭 미션: 런던 박물관에서      │
│  새 동료를 만났습니다          │
│                              │
│  ┌─ 루디 ─────────────────┐  │
│  │ Hello! I'm Rudy.      │  │
│  │ What's your name?      │  │
│  └────────────────────────┘  │
│                              │
│  💡 추천 답변:                │
│  ┌─────────────────────────┐ │
│  │ "My name is ___."      │ │
│  │ "Hello, I'm ___."      │ │
│  └─────────────────────────┘ │
│                              │
│       [🎤 말하기]             │
│                              │
│  또는 [⌨️ 키보드로 입력]      │
│  (음성 답변 시 XP 1.5배!)     │
│                              │
├─────────────────────────────┤
│  12/24 발화 완료              │
└─────────────────────────────┘
```

### 동작 흐름

1. 상단에 미션 상황 설명 (nativeLang으로)
1. GPT-4o가 루디(NPC) 역할로 첫 대사를 보냄 (targetLang으로)
1. 유저가 마이크로 음성 응답 → Azure Speech로 텍스트 변환 → GPT-4o에 전달
1. GPT-4o가 다음 대사로 응답 (대화 계속)
1. 5~6번 대화 주고받기 = 유저 8~10문장 발화
1. GPT가 대화 종료 시점을 판단하면 “미션 완료!” 표시
1. STEP 4로 전환

### 핵심 규칙

- **음성 우선**: 기본 답변 방식은 마이크 버튼. “키보드로 입력” 버튼은 작게 아래에 배치
- **음성 보너스**: 음성으로 답하면 해당 답변 옆에 “🎤 XP 1.5배!” 작게 표시
- **추천 답변 표시**: A1~A2에서는 추천 답변 2~3개를 항상 보여줘
- **추천 답변 탭 동작**: 추천 답변을 탭하면 텍스트가 입력되는 게 아니라, 그 문장을 소리내어 읽도록 마이크가 활성화됨. 즉 탭해도 반드시 말해야 함
- **실시간 교정**: GPT가 문법 실수를 자연스럽게 교정 (예: 유저가 “I name is…“이라고 하면 루디가 “Nice! By the way, we say ‘My name is’ 😊”)
- **BGM 음소거**: STEP 3 전체에서 BGM 꺼줘 (마이크 계속 사용하니까)
- **루디 대사 TTS**: 루디의 대사는 자동으로 TTS 재생 (targetLang 음성)

### GPT-4o 시스템 프롬프트 템플릿

Day 데이터의 missionTalk.gptPrompt를 시스템 프롬프트로 사용해. 거기서 {targetLang}을 유저의 targetLang으로 치환해줘.

추가로 모든 프롬프트에 이 공통 규칙을 붙여줘:

```
Common rules for all Mission Talk:
- Speak ONLY in {targetLang}
- Keep sentences short and simple for the current difficulty level
- If the player makes grammar mistakes, gently model the correct version in your next line (don't say "wrong" or "incorrect")
- Encourage full sentences, not single words
- After 5-6 exchanges, naturally wrap up the conversation
- After EVERY response, append hidden JSON on a new line:
  [EVAL]{"sentenceCount": N, "grammarNote": "...", "shouldEnd": false}[/EVAL]
```

프론트에서 [EVAL]…[/EVAL] 사이 JSON을 파싱해서 sentenceCount 추적하고, shouldEnd가 true면 대화 종료 처리해줘. grammarNote는 나중에 미션 완료 리포트에 활용할 수 있도록 저장해둬.

### 대화 종료 처리

shouldEnd: true 가 오면:

1. 루디의 마지막 대사 표시 + TTS 재생
1. “🎉 대화 미션 완료!” 배너 표시 (2초)
1. 자동으로 STEP 4로 전환

## STEP 4: 복습 챌린지 (Quick Review)

### 화면 구성

```
┌─────────────────────────────┐
│  Day 1 · 만남 인사            │
│  STEP 4/4 복습 챌린지         │
│  ████████░░                  │
├─────────────────────────────┤
│                              │
│  🦊 "마지막! 빠르게 복습하자!" │
│                              │
│  Q 1/5                       │
│  ┌───────────────────────┐   │
│  │ 🎤 이 문장을 말해보세요  │   │
│  │                       │   │
│  │ "Hello, nice to       │   │
│  │  meet you."           │   │
│  │                       │   │
│  │    [🎤 말하기]         │   │
│  └───────────────────────┘   │
│                              │
│  ⏱️ 15초                     │
│  ████████░░░░░░              │
│                              │
├─────────────────────────────┤
│  20/24 발화 완료              │
└─────────────────────────────┘
```

### 동작 흐름

1. 5문제 빠르게 진행
1. 각 문제마다 15초 카운트다운 타이머 (상단 프로그레스 바)
1. 문제 유형:
- speak: 화면에 문장 표시 → 유저가 마이크로 읽기 → 발음 체크
- fill_blank: 빈칸 선택 → 정답 후 speakAfter (전체 문장 말하기)
1. 15초 안에 답하면 정답 처리 + 점수 표시
1. 시간 초과 → “시간 초과!” 표시 후 정답 보여주고 자동으로 다음 문제
1. 5문제 완료 → 미션 완료 화면

### 어제 복습 vs 오늘 문제 구분

Day 데이터의 review.questions에 isYesterdayReview: true가 있는 문제는 어제 복습 문제야.

- Day 1에는 isYesterdayReview 문제가 없음 (첫날이니까)
- Day 2부터 2개가 어제 복습, 3개가 오늘 문제
- 어제 복습 문제에는 “📌 복습” 작은 태그 표시

## 미션 완료 화면

### 화면 구성

```
┌─────────────────────────────┐
│                              │
│        🎉 미션 완료!          │
│                              │
│  📊 오늘의 성과               │
│  ──────────────────          │
│  🗣️ 말한 문장: 22문장         │
│  🎯 평균 발음: 82%            │
│  📝 새 표현: 3개              │
│  ⭐ 기본 XP: +100             │
│  🎤 음성 보너스: +50           │
│  ──────────────────          │
│  총 XP: +150                  │
│                              │
│  🦊 "오늘도 수고했어, 파트너!  │
│   내일은 '출신 말하기'를       │
│   배울 거야. 내일 봐!"        │
│                              │
│  [홈으로]                     │
│                              │
└─────────────────────────────┘
```

### 성과 계산

- 말한 문장 수: 전체 STEP에서 마이크로 말한 횟수 합산
- 평균 발음: STEP 1 + STEP 4의 발음 점수 평균
- 새 표현: STEP 1에서 연습한 문장 수
- 기본 XP: Day 데이터의 rewards.xp
- 음성 보너스: 키보드 안 쓰고 전부 음성으로 했으면 rewards.bonusAllVoice
- 발음 보너스: 평균 발음 90% 이상이면 rewards.bonusPronunciation

### 완료 시 저장

```javascript
// AsyncStorage 업데이트
progress.completedDays.push(currentDay.id);
progress.todayCompleted = true;
progress.stats.totalSentencesSpoken += todaySentenceCount;
progress.stats.totalDaysCompleted += 1;
// 다음 Day 해금
progress.currentDayIndex += 1;
// 스트릭 업데이트
progress.stats.currentStreak += 1;
// XP는 기존 XP 시스템에 추가
```
