## Fixes needed

### 1. Add retry logic for GPT-4o API calls

When the API call fails, automatically retry up to 3 times before showing an error:

```javascript
async function callGPT(messages, retries = 3) {
  for (let i = 0; i < retries; i++) {
    try {
      const response = await fetch('https://api.openai.com/v1/chat/completions', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'Authorization': `Bearer ${OPENAI_API_KEY}`,
        },
        body: JSON.stringify({
          model: 'gpt-4o',
          messages: messages,
          max_tokens: 500,
        }),
        // Add timeout
        signal: AbortSignal.timeout(15000), // 15 second timeout
      });
      
      if (!response.ok) {
        throw new Error(`API error: ${response.status}`);
      }
      
      const data = await response.json();
      return data.choices[0].message.content;
    } catch (error) {
      console.log(`[MissionTalk] GPT attempt ${i + 1} failed:`, error);
      if (i < retries - 1) {
        // Wait before retry (exponential backoff)
        await new Promise(resolve => setTimeout(resolve, 1000 * (i + 1)));
      }
    }
  }
  // All retries failed
  return null;
}
```

Only show the connection error message after all 3 retries fail:

```javascript
const gptResponse = await callGPT(conversationHistory);

if (gptResponse) {

```
