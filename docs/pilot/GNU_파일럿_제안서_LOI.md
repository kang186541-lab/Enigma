# [제안서] AI 말하기 트레이너 "LinguaAI" 강좌 파일럿 (4~6주, 무상)
**수신:** 경상국립대학교 어학교육원 / 교양영어 담당 교수님
**제안:** LinguaAI 팀 (경상국립대 모두의 창업 참가팀)
**소요 비용:** 무상 (파일럿 기간 전체) · **학생 부담:** 없음 · **설치:** 불필요(웹 링크)

---

## 1. 한 문단 요약
LinguaAI는 **"말해야 진행되는"** AI 회화 트레이너입니다. 학생이 매일 5문장을 소리 내어 말하면 **AI가 음소 단위로 발음을 진단**하고 모국어로 교정 피드백을 줍니다. 교수님께는 **분반 대시보드**로 학생별 연습량·발음 점수·이탈 위험을 한눈에 제공합니다. 본 파일럿은 1개 분반(20~60명)에서 4~6주간 **"수업 외 말하기 연습량을 늘리면 무엇이 달라지는가"**를 함께 측정하는 실증입니다.

## 2. 왜 필요한가 (근거)
- 한국 영어교육 지출은 과목 1위지만, EF 영어능력지수는 **세계 50위로 2년 연속 하락** — "읽고 외우는데 말은 못 하는" 구조적 격차.
- 교육용 앱의 30일 잔존율은 업계 평균 **~2%**: 숙제식 앱은 학생이 금방 떠납니다. LinguaAI는 추리 스토리·게임 메커닉으로 "내일 또 말하러 오게" 설계되었습니다.
- 수업 시간엔 1인당 발화 기회가 수 분에 불과 — **수업 밖 발화량을 늘리는 보조 도구**가 필요합니다.

## 3. 파일럿 설계 (교수님 부담 최소화)
| 항목 | 내용 |
|---|---|
| 대상 | 1개 분반 20~60명 (교양영어/회화 강좌) |
| 기간 | 4~6주 |
| 학생 할 일 | 첫 주 오리엔테이션 5분(수업 코드 입력) + **매주 필수 미션**(데일리 말하기 2회 + 상황 회화 NPC 1회, 회당 ≈5분) + **선택 게임 미션**(추리 스토리·방탈출, 자율) |
| 교수님 할 일 | 주 1회 대시보드 확인(선택) + 종료 후 15분 피드백 인터뷰 |
| 우리가 제공 | 수업 코드 발급, 학생 온보딩 안내문, 주간 리포트(PDF), 기술 지원 |
| 개인정보 | 학생 동의 기반, 이메일 마스킹(예: k***@gnu.ac.kr), 음성 원본 미보관, 종료 후 데이터 삭제 요청 가능 |

## 4. 함께 얻는 것
**교수님/기관:**
- 분반 대시보드: 학생별 연습 횟수·연속 학습일·발음 점수 추이·**이탈 위험 알림**(3일 이상 미접속)
- 종료 리포트: 주차 완료율, 주차 지속률, 발화량 변화, 발음 수행 밴드 이동 (강의 성과 자료로 활용 가능)
**LinguaAI 팀:**
- 실사용 리텐션·학습효과 데이터 (창업 검증용, 익명 통계만 대외 사용)

## 5. 측정 지표 (종료 리포트에 포함)
6주 동안 **"수업 밖 말하기 과제를 측정 가능하게 만들 수 있는가"**를 5개 지표로 확인합니다.
1. **주차 완료율:** 매주 필수 미션 중 2개 이상 완료한 학생 비율 (목표 60%+)
2. **발화량:** 학생 1인당 6주 누적 발화 시도 횟수(목표 30회+) · 활동일수
3. **주차 지속:** 1주차 참여 학생 중 4·6주차에도 활동한 비율 (교육앱 평균 대비)
4. **발음 수행 변화:** 동일 과제 재수행 시 발음 수행 밴드(개선필요→코칭→통과) **이동** 비율 — *절대 점수가 아닌 "이동"으로 보고*
5. **교수님 재사용 의향:** "다음 학기에도 과제로 쓸 의향" (5점 척도 + 주관식)

## 6. 일정 (제안)
주0: 코드 발급·오리엔테이션 자료 전달 → 주1: 학생 가입(수업 코드 입력) → 주1~5: 자율 연습 + 주간 리포트 → 주6: 종료 설문·리포트·인터뷰.
